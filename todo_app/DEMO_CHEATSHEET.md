## デモ前の準備

```bash
# 1. calculator_appフォルダの中身を削除して、ZIPの中身を展開
# 2. セットアップ
cd /Users/harukiuriu/Desktop/calculator_app
flutter create . --project-name calculator_app
flutter pub get

# 3. Gitリポジトリ初期化（コミットデモ用）
git init
git add .
git commit -m "initial commit"

# 4. エミュレータを起動してアプリを実行
flutter run

# 5. Claude Codeを別ターミナルで起動
claude
```

**画面配置:** ターミナル（Claude Code）を左、エミュレータを右に並べる
